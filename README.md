## 1. Island-OS 是什麼？

Island-OS Whitepaper Series 是一組針對 **島嶼型國家** 設計的  
多域韌性與防禦模組框架，涵蓋：

- 空域拒止  
- 電磁防護  
- 地面自癒防線  
- 生態／海洋韌性  
- 社會與系統黏稠度  

本 repo 中的每一篇白皮書，都可以視為 Island-OS 的一個「模組」。
